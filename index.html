My work spans virtual and augmented reality (VR/AR), neurotechnology, and cybersecurity, where I aim to explore the intersection of innovation and practical applications. Work

I have developed a diverse technical background through internships, research, and collaborative projects, many of which are detailed in my resume. You can also explore all my projects showcased below—just slide through to take a closer look!

Mixed Reality Interactions using a Depth Camera

Developed a real-time system enabling interactions between virtual objects and the physical environment. Using an Intel RealSense D435 depth camera, I implemented a voxel-based approach that allowed virtual elements to interact with real-world surfaces. The system was integrated with an Oculus Quest 2 headset, demonstrating effective real-time interaction with physical objects.

Cooperative Virtual Reality Game

As part of a development team, I contributed to creating a cooperative VR game using Unity and C#. The game focused on collaboration, immersive combat, and resource management, with interactive crafting and modular weapon systems. I worked on the immersive control system and gameplay mechanics, ensuring a highly interactive and engaging experience.

Computer Vision HandMouse

Designed a real-time hand tracking and gesture recognition system using Python, Google’s Mediapipe, OpenCV, and PyCaw. This system allows users to control Windows OS using hand gestures, designed primarily for accessibility. I utilized multi-threading to optimize gesture recognition, improving real-time performance.

Capstone 2D Game Project

Collaborating in a team, we developed a 2D platformer combat game using Unity and C#. My contributions included implementing a grappling hook system, dynamic dice-based stat modifiers, and integrating Agile project management tools like Trello and Git for efficient task management.

Augmented Reality game

Developed an augmented reality (AR) game featuring a 3D maze overlaid on the real world using a QR code. Built with Unity and Visual Studio, I utilized C# and raycasting techniques for gameplay logic, allowing users to physically navigate and interact with the maze in a real-world environment.

iOS Application

Developed a task management iOS application using Xamarin and C# in Visual Studio. The app allows users to create, edit, prioritize, and track tasks efficiently. The application was built cross-platform, enabling it to run seamlessly on both iOS and Android devices.

Virtual Reality Avatar

Created a VR avatar for use in VR applications and games. The avatar was designed from scratch using Blender and Unity, with expertise in 3D modeling, bone rigging, texturing, shading, and asset creation. The avatar can be integrated into various VR environments for immersive experiences.

Electromagnet Controller

Developed a device using three electromagnets to manipulate ferrofluid in water, controlled by an Arduino board. The device creates mesmerizing visual effects by moving the ferrofluid in response to specific timing and voltage control through custom circuitry and cable soldering.
